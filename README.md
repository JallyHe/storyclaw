# StoryClaw

AI-powered screenplay editor with Agent/Skill system.

## Releases

All binary releases are available in the [Releases](https://github.com/JallyHe/storyclaw/releases) section.

- **Windows**: Download `StoryClaw Setup X.X.X.exe` from the latest release
- **Auto-update**: The application automatically checks for updates from GitHub Releases

## Project Structure

- Source code is not included in this repository
- Only compiled binaries and update manifests are distributed
- Updates are managed through `latest.yml` in each release
