# 连续性校验

**定义**：连续性校验排查跨场次的连戏错误——时间线、空间位置、道具状态、人物认知/伤情、信息掌握是否前后一致。

**核心约束**：
- 维护时间线与信息位的全局一致（谁在何时知道什么）。
- 道具、伤情、外观等状态跨场次要连得上。
- 矛盾处标注涉及的场次，给出修正方案。

**在创作中的应用**：逻辑校对专家用来出具连续性问题清单。

**自检清单**：
- [ ] 时间线一致
- [ ] 信息位一致
- [ ] 状态连得上
- [ ] 矛盾标注场次
