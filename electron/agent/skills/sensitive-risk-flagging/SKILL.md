# 敏感风险标记

**定义**：敏感风险标记快速扫描剧本中可能引发争议或法律风险的敏感点——影射真人、特定地域/行业/群体的负面呈现、历史事件处理等。

**核心约束**：
- 标记敏感点并说明潜在风险类型（名誉/地域/群体/历史等）。
- 区分"必须处理"与"提请注意"。
- 给出脱敏或模糊化的处理建议。

**在创作中的应用**：合规风控专家用来产出敏感点清单。

**自检清单**：
- [ ] 标明风险类型
- [ ] 区分必处理/提请注意
- [ ] 给脱敏建议
- [ ] 覆盖真人影射
