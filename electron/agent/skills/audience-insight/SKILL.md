# 核心洞察

**定义**：核心洞察是一句直指目标受众内心、能引发强烈共鸣的情感/心理真相——它是作品打动人的根本支点。

**核心约束**：
- 洞察要真实、具体、戳中痛点或痒点。
- 洞察必须能落到主角的永久困境上。
- 一部作品聚焦一个核心洞察，不贪多。

**在创作中的应用**：阶段1 提炼，作为主题与主角困境的根。

**自检清单**：
- [ ] 真实戳中受众
- [ ] 可落到主角困境
- [ ] 单一聚焦
