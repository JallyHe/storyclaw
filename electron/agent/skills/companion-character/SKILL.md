# 伙伴角色

**定义**：伙伴角色陪伴主角、在功能上衬托或补足主角，常承担情感支持、信息提供或反衬对照的作用。

**核心约束**：
- 伙伴要有独立的小目标，不能纯工具人。
- 通过与主角的差异反衬主角特质。
- 伙伴与主角的互动模式应稳定且可辨识。

**在创作中的应用**：阶段2 写入关系与互动模式；阶段3-6 落实其衬托与情感功能。

**自检清单**：
- [ ] 有独立小目标
- [ ] 差异反衬主角
- [ ] 互动模式稳定
