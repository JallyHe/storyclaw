# 终结情节

**定义**：终结情节（B）结束旧情境、释放积累的戏剧压力，同时种下下一个 C 的种子，完成情境之间的交接。

**核心约束**：
- B 必须真正解决当前 C 的核心张力，让观众的追问得到回答。
- 解决要付出代价，不能无损通关。
- 理想的 B 在收束旧 C 的同时埋下新 C 的引信。

**在创作中的应用**：阶段2 Step 2B 为每个 C 设计 B；阶段3 在梗概中标注 B 节点。

**自检清单**：
- [ ] 核心张力被真正解决
- [ ] 付出了代价
- [ ] 是否播下新种子（如适用）
