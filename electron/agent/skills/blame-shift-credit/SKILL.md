# 甩锅归功

**定义**：甩锅归功：把主角的过错与负面后果合理转嫁/稀释（甩锅），把功劳与正面结果归于主角（归功），从而保护主角的道德吸引力。

**核心约束**：
- 主角的缺陷可以存在，但其恶果不应由无辜者承担到令观众反感的程度。
- 关键善果应让主角的选择起决定作用，使观众愿意认同。
- 甩锅要合理、不刻意，否则会显得主角虚伪。

**在创作中的应用**：阶段3-5 设计主角行为与后果分配时使用，尤其在道德灰色的桥段。

**自检清单**：
- [ ] 主角缺陷不致观众反感
- [ ] 关键善果归于主角选择
- [ ] 甩锅自然不刻意
