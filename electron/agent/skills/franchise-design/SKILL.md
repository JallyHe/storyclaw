# 系列化设计

**定义**：系列化设计为作品规划可持续的续作与衍生结构——哪些线索留给续集、哪些角色可独立成篇、世界规则如何支撑多部作品。

**核心约束**：
- 在不损害单部完整性的前提下，预留续作的种子。
- 设计可独立承载新故事的角色与支线。
- 世界规则要为后续留出扩展余地而非自我封死。

**在创作中的应用**：IP衍生专家用来规划系列蓝图与续作种子。

**自检清单**：
- [ ] 单部仍完整
- [ ] 预留续作种子
- [ ] 有可独立成篇的角色
- [ ] 世界规则可扩展
