# 情节触发角色

**定义**：情节触发角色主要通过行动推动事件发生（常见为反派或搅局者），是 D（情境铺垫）与复杂化的重要来源。

**核心约束**：
- 其行动要有自洽的动机，不能为了推进剧情而行动。
- 触发的复杂化应来自其性格，而非编剧的方便。
- 强度要与主角匹配，过弱无张力，过强则碾压。

**在创作中的应用**：阶段2-3 用其行动设计 D 与 A 的复杂化；阶段5-6 落实其压迫感。

**自检清单**：
- [ ] 动机自洽
- [ ] 复杂化源自其性格
- [ ] 强度与主角匹配
