# 动作催化（对白配动作）

**定义**：动作催化把「站着说话」的静态对白转化为在动作中进行的对白，让台词与肢体、场面调度结合，增强画面感与张力。

**核心约束**：
- 给对白配上有意义的动作，而非无意义的走位。
- 动作应反映或反衬台词的潜台词。
- 优先「展示」而非「告知」。

**在创作中的应用**：阶段6 R10，提升对白的画面化程度。

**自检清单**：
- [ ] 对白配有意义动作
- [ ] 动作呼应潜台词
- [ ] 展示优先于告知
