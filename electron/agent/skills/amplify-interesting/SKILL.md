# 放大有趣

**定义**：放大有趣：以当前故事情境为过滤器，把服务于情境、张力强的有趣点放大展开；不服务于情境的趣味即使好笑也要修剪。

**核心约束**：
- 「有趣」必须挂靠在某个 C 上才放大。
- 游离于情境之外的段子/奇观一律压缩或删除。
- 放大手段不得削弱情境压力（与 pressure-escalation 协同）。

**在创作中的应用**：阶段3 整合梗概、阶段4 分场时筛选与放大看点。

**自检清单**：
- [ ] 被放大的趣味都服务于 C
- [ ] 游离趣味已修剪
- [ ] 放大未削弱压力
