# 张力释放（喘息）

**定义**：张力释放是在持续高压中插入短暂的喘息或幽默，调节观众情绪节奏；但根本压力不被削弱（与 pressure-escalation 协同）。

**核心约束**：
- 喘息要短，且不解决核心张力。
- 喘息内容最好仍与情境/人物相关。
- 喘息后要迅速把压力重新拉回甚至拉高。

**在创作中的应用**：阶段6 R9，精调高压段的呼吸感。

**自检清单**：
- [ ] 喘息短促
- [ ] 不解决核心张力
- [ ] 之后压力回升
