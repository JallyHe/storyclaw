# 内容合规红线

**定义**：内容合规红线审查剧本是否触碰内容监管的红线（暴力、违法示范、价值导向、敏感议题等），并评估过审风险。

**核心约束**：
- 按风险等级（高/中/低）标注问题点与具体位置。
- 不只指出问题，给出合规的替代写法。
- 把握尺度，避免过度自我审查扼杀创作。

**在创作中的应用**：合规风控专家用来出具合规风险报告与改写建议。

**自检清单**：
- [ ] 风险分级标位置
- [ ] 给合规替代写法
- [ ] 尺度得当
- [ ] 覆盖价值导向
