# 奖赏角色

**定义**：奖赏角色是主角欲望所指向的对象之一，代表一种「得到即满足」的情感奖赏（如爱慕对象、认可来源），其可得与否构成张力。

**核心约束**：
- 奖赏的获得必须有门槛，唾手可得就失去张力。
- 奖赏角色应有自身意志，而非被动的奖品。
- 靠近/远离奖赏的过程要与主线情境同频。

**在创作中的应用**：阶段2 确立奖赏关系；阶段3-6 用其可得性调节张力。

**自检清单**：
- [ ] 获得有门槛
- [ ] 有自身意志
- [ ] 与主线同频
