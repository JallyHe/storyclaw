# 问答角色反转

**定义**：问答角色反转打破机械的一问一答：让被提问者反过来掌控对话，或答非所问，以制造博弈感与人物主动性。

**核心约束**：
- 识别并改写呆板的「采访式」对白。
- 让回答带出反问、回避或反将一军。
- 反转要符合角色的信息位与意图。

**在创作中的应用**：阶段6 R6，消灭采访式对白。

**自检清单**：
- [ ] 采访式对白已改写
- [ ] 回答有博弈
- [ ] 符合角色信息位
