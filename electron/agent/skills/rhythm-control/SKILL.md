此 skill 定义和阐述对白优化原则之一：**节奏控制**（Rhythm Control）。

## 核心原则

**对话的速度和节奏不是装饰——它是控制观众呼吸的工具。不同的戏剧场景需要不同的节奏：紧张场景用短句和快速切断，抒情场景用稍长的句子和留白，关键揭示前用铺垫和延迟。**

## 三种节奏模式

### 紧张场景：快切 + 短句 + 打断

紧张场景中的对话应该像乒乓球——球速越来越快，回球的间隙越来越短。

要素：
- 每一句台词的文字长度控制在 15 字以内
- 角色之间频繁互相打断——不是等对方说完再接
- 减少动作描写在紧张段中的占比——让对话本身驱动速度
- 在"最快的时刻"插入紧张断点（[[tension-break]]）

```
"你知道了？"
"知道什么。"
"别装。"
"我没装。"
（沉默——被打断）
"你在发抖。"
"冷。"
```

### 抒情场景：长句 + 停顿 + 留白

抒情场景中的对话不是"慢"——是给语言足够的空间去沉下去。观众需要时间去感受每一句的重量。

要素：
- 适当使用稍长的句子（20-30 字），让语言有展开的余地
- 在关键句子的前后使用动作或空白制造"呼吸"
- 不要每一句都填满——留白也是台词的一部分
- 不要在抒情段落中突然插入快节奏的冲突打断——会让情绪断裂

### 关键揭示前：铺垫 + 延迟 + 制造期待

揭示不是"突然说出来"——是在说出之前让观众感到"有什么东西要被说出来了"。这个"延时"制造的期待，比答案本身更重要。

要素：
- 在揭晓信息之前，先展示角色的"迟疑"或"组织语言"的瞬间
- 用一个微小的动作延迟信息的释放（喝口水、看了一眼窗外、翻了一下手里的东西）
- 揭示后的第一秒——不要立刻接下一句台词。让信息在空中悬一瞬

```
"你知道是谁做的。"
（他没有回答。）
"告诉我。"
（他把杯子放下来，杯底碰到桌面——很轻。然后他抬起头。）
"是我。"
```

## 节奏的全局控制

整场戏的节奏不是匀速的——它应该有一个加速/减速的弧：

- **开场**：中性节奏，铺垫情绪基调
- **中段**：加速或减速（取决于戏剧目的——是升温还是沉入）
- **转折点前**：加速到一个顶点——然后暂停（[[tension-break]]）
- **结尾**：要么以快切收尾（冲击力），要么以留白收尾（余味）

## 常见错误

### × 全程高速
紧张的对话从头快到尾——观众会麻木。没有速度变化的"紧张"在第三十秒之后就开始失效。

### × 在抒情段落里突然插入快节奏
"我想告诉你——这些年我一直在等你。" → 然后立刻接一个动作打断。这是浪费了情感累积的时机。
