# 节奏控制

**定义**：节奏控制通过长短句交替、停顿、留白与信息密度的起伏，塑造对白与场面的呼吸节奏。

**核心约束**：
- 长短句交替，避免节奏单一。
- 在情感高点用停顿/留白放慢，在推进段加快。
- 整场/整集的节奏要有起伏曲线。

**在创作中的应用**：阶段6 R11，统调台词与场面的节奏。

**自检清单**：
- [ ] 长短句交替
- [ ] 高点有留白
- [ ] 整体节奏有起伏
