# 压力只增不减

**定义**：在一个情境被解决之前，其内部的戏剧压力必须持续递增，不能被任何形式的「缓解」提前削弱。

**核心约束**：
- 情境解决前禁止出现让观众松一口气的实质性缓解。
- 允许节奏上的短暂喘息（tension-break），但不得降低根本压力。
- 每个 A 情节结束都应让总压力更高。

**在创作中的应用**：阶段2 设计 DAB、阶段3-5 落地时，逐段核对压力曲线是否单调上升。

**自检清单**：
- [ ] 解决前无实质缓解
- [ ] 喘息不降低根本压力
- [ ] 压力曲线单调上升
