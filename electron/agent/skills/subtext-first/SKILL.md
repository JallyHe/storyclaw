# 潜台词优先

**定义**：潜台词优先：人物尽量不直接说出真实意图与情绪，而是通过表层话语暗示底层欲望，让观众去解读。

**核心约束**：
- 把直白的情绪/意图改写为间接表达。
- 表层话语与底层欲望之间要有张力。
- 关键信息可藏于潜台词，但不可让观众完全 get 不到。

**在创作中的应用**：阶段6 R2，是对白优化的核心一轮。

**自检清单**：
- [ ] 直白表达已改间接
- [ ] 表层/底层有张力
- [ ] 观众仍能解读出潜台词
