此 skill 定义和阐述"社会洞察"（Social Insight）——分析故事与当下社会流行的认知和价值观念之间的契合点。

## 核心定义

**社会洞察回答的问题是：这个故事跟当下社会最新流行的认知、价值观念、焦虑、不安和向往——有什么高度契合的地方？它反映了目标观众最新的思潮、观念变迁和时代情绪。十年前可能不成立，现在成立——这中间的差异就是社会洞察要找到的东西。**

观众洞察是"人类永远需要被爱"——社会洞察是"但这一代人对'被爱'的定义已经变了——他们不再觉得'牺牲自己为你好'是爱，他们要的是'你尊重我的边界'。这个故事中的爱情模式精准击中了这个新观念。"

## 与观众洞察的严格区分

| 问题 | 属于 | 示例 |
|------|------|------|
| 为什么人类需要这个故事？ | 观众洞察 | "因为人类永远需要被看见和被认可" |
| 为什么这个故事现在成立？ | 社会洞察 | "因为社交媒体时代制造了一种新的'被忽视'——每个人都在展示自己，但没有人真正被看见" |
| 角色在满足什么心理需求？ | 观众洞察 | "主角的逆袭满足了观众在现实中缺乏的掌控感" |
| 逆袭的"方式"为什么变了？ | 社会洞察 | "十年前的主角靠努力和贵人翻身，现在的主角靠打破规则——因为这一代人对'努力就能成功'的信仰已经崩塌了" |

## 社会洞察的扫描维度

从以下维度逐一扫描当下社会的观念变迁，找到故事与之契合的点：

### 1. 权力观念的变迁
- 从服从权威 → 质疑权威：主角不再服从师长/老板/体制，而是用自己的方式建立新秩序
- 从等级固化 → 阶级流动焦虑："努力就能上去"的信念在崩塌——逆袭叙事的形式在改变
- 从"被保护" → "自己保护自己"：女性角色不再是等待拯救的对象

### 2. 亲密关系观念的变迁
- "为你好"被重新定义：牺牲不再是爱的证明，边界感才是
- "灵魂伴侣"的瓦解：这一代人越来越不相信"有一个人是为你而生的"
- "治愈"关系取代"拯救"关系：不是"我来救你"，是"我看见你的伤口，我不走"

### 3. 成功定义的变迁
- 从"拥有"到"成为"：物质成功不再是终极目标——自我实现成为新的成功叙事
- 从线性奋斗到多元路径：这一代人不再相信好好读书→好工作→好人生的单一路径
- "躺平"与"上岸"的拉扯：角色在"我要拼命"和"我够了"之间摇摆

### 4. 性别观念的变迁
- 女性自主权的凸显：女性角色的欲望、缺点、野心——不再被道德审判
- 男性气质的重新定义：脆弱、流泪、退缩——男性角色被允许"不强大"
- 关系中的权力对等：观众不再接受单方面付出的爱情模式

### 5. 焦虑焦点
- 经济安全感：对"下个月还能不能维持现在的生活"的深层不安
- 身份焦虑：在信息过载的世界上"我是谁"比以前更难回答
- 孤独感：社交媒体让人人都可以被触及，但触及之后的深度在消失

## 分析方法

### 第一步："为什么是现在"
这个故事的核心设定或核心冲突——如果放在十年前，它还成立吗？如果不成立——什么变了？这个"变了的东西"就是社会洞察的入口。

### 第二步：找到"新观念"的锚点
在故事中找到——哪个角色的哪句台词、哪个选择、哪个关系模式——是在反映一种"十年前可能不会被写出来"的新观念？这个锚点就是社会洞察的核心论据。

### 第三步：观众的现实对照
目标受众在现实中正在经历什么与这个故事高度相关的焦虑或向往？这个故事是如何精准地触碰到这个焦虑/向往的？

## 输出格式

**1. 核心社会契合点**：[一句话] 这个故事在 [什么方面] 精准击中了当下 [哪群人] 正在经历的 [什么变迁/焦虑/向往]

**2. 观念的"以前 vs 现在"**：
- 以前的观念 / 常见的处理方式是：[X]
- 现在的观念 / 这个故事的处理方式是：[Y]
- 这个转变说明什么：[社会心理的底层变化]

**3. 故事中的社会锚点**：[从故事中提取至少 2 个具体元素（角色选择、对白、关系模式、情节设定）]，逐一说明它们如何反映了当下的新观念

**4. 时代情绪连接**：这个故事与当下受众的哪一股时代情绪同频？[不安 / 愤怒 / 治愈渴望 / 重来一次的幻想 / 逃离的冲动……]

**5. 时效性评估**：这个社会洞察的保质期有多长？[这股思潮正在上升期 / 已经到达高峰 / 正在消退]

## 与同组概念的关系

- 与 [[audience-insight]] 严格区分 —— 观众洞察是 timeless 的人性需求，社会洞察是当下的观念变迁。两者回答不同时间尺度的问题
- 与 [[target-audience]] 配合 —— 社会观念的变迁可能会改变目标受众的范围
- 与 [[genre-classification]] 配合 —— 社会洞察可能影响题材的"时效吸引力"
