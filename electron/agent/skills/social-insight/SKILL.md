# 社会洞察

**定义**：社会洞察是作品所呼应的更大的社会议题或时代情绪，为作品提供话题性与超越个体故事的共鸣。

**核心约束**：
- 社会洞察要通过具体故事自然承载，不能说教。
- 与核心洞察呼应而非割裂。
- 把握分寸，避免敏感雷区与生硬贴标签。

**在创作中的应用**：阶段1 提炼，增强卖点与话题潜力。

**自检清单**：
- [ ] 通过故事自然承载
- [ ] 与核心洞察呼应
- [ ] 分寸得当
