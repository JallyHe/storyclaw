此 skill 定义和阐述故事架构原则之一：**人物声音差异化**（Character Voice Differentiation）。

## 核心原则

**如果遮住角色名字读一段对话，观众无法分辨出每句台词是谁说的——这段对话没有完成它的工作。每个角色应该拥有独特的词汇选择、句子长度、说话节奏和语言习惯。**

这不是一个关于"风格"的建议——这是关于信息效率和角色辨识度的硬性约束。在一场多人对话中，如果所有角色"听起来一样"，观众的大脑必须在每一句台词上额外处理"这是谁说的"——这会消耗注意力，让观众无法同时关注潜台词和情节推进。

## 差异化维度

人物声音的差异可以——也应该——在以下多个维度上同时建立：

### 1. 句子长度
- 短句角色：精确、直接、不解释。每句话平均 5-10 个字。
- 长句角色：喜欢展开、喜欢限定条件、喜欢用从句。每句话可能跨越多行。
- 混合型：在人多的场合短，在独处或信任的人面前长。

### 2. 词汇选择
- 词汇域：使用哪个领域的词汇？（学术词汇 / 街头俚语 / 商业术语 / 日常口语）
- 抽象度：喜欢具体描述还是抽象概括？
- 情感词汇：是否使用表达情感的词汇？还是回避？

### 3. 句法习惯
- 主谓宾顺序，还是倒装？
- 喜欢用"但是"转折，还是喜欢用"而且"叠加？
- 喜欢下结论，还是喜欢提出问题？

### 4. 节奏标记
- 说话中有没有"嗯""那个""就是"这种填充词？
- 喜欢在句末上扬（不确定）还是下沉（确定）？
- 有没有属于自己的口头禅或反复出现的句式？

### 5. 对话策略
- 面对质问：直接回答 / 反问 / 转移话题 / 沉默
- 面对赞美：接受 / 否认 / 自嘲 / 转移给他人
- 面对沉默：填补沉默 / 享受沉默 / 被沉默逼迫着说出不该说的话

## 差异化测试

写一段三个角色的对话。然后遮住所有的角色名。大声读出来。让另一个人读——或者录下来自己听。

**测试问题：**
- 每一句的说话者能被准确识别吗？
- 有没有哪两句——如果交换说话者——听起来也毫无违和感？
- 在某一句中，有没有一个"只有这个角色会这么说"的特定措辞？

如果对以上任一问题的答案不理想——回去重新设计这些角色的语言特征。

## 差异化不是刻板化

人物声音的差异化并不意味着每个角色都要挂着一句口头禅或者一个奇怪的语法错误。最高级的差异化是——

**每个人物的大脑运作方式不同，所以他们组织语言的方式不同。**

一个习惯"分析"的角色看到一个问题会先说"目前有三个因素"——他的对白自带编号。一个习惯"感受"的角色看到同样的问题会说"我不知道为什么，但就是不对劲"——她的对白拒绝对情绪进行编号。

这不是口头禅。这是思维方式的差异在语言上的折射。这才是真正让人物声音不可互换的原因。

## 常见错误

### × 把所有角色都写得像作者自己
创作者通常有一种自然的语言习惯——自己的语言习惯。然后他/她把这种习惯分给了所有角色。结果就是：所有角色都在替作者说话。

### × 靠"标签"来区分，而不是靠"思维"
"这个角色每句话结尾都加'对吧'"——这是标签。标签容易写，也容易被观众识破为"作者在装神弄鬼"。

### × 在功能对白上发力，在关键对白上统一
许多创作者在最不重要的旁白/过渡对白上做了差异化——而在真正的关键时刻，所有角色突然统一成了一种"剧情需要的标准声音"。关键对白才是声音差异化的最高考场——角色的本质在压力下才能真正暴露。

## 与[[character-situation]]的关系

人物声音差异化是角色情境（character-situation）在语言层面的表达。谢尔顿的语言中的逻辑强迫和社交盲区——不是作者给他贴的标签，是他的角色情境（超常智商 + 零社交直觉）在每一句台词中的自然折射。好的差异化不是"设计的"，是"从角色底色中推导出来的"。

## 与同组技巧的关系

- 与 [[qa-role-reversal]] 配合 —— 不同角色的"翻转模式"选择本身就是人物声音差异化的体现
- 与 [[subtext-first]] 配合 —— 人物掩饰真实想法的方式（有的是沉默、有的是过度解释、有的是攻击）是声音差异化的深层维度
- 与 [[callback-echo]] 配合 —— 同一句台词在不同角色口中被复述时，声音差异被最大化
- 与 [[meta-commentary]] 配合 —— 每个角色的元评论风格不同：有人自嘲、有人点评他人、有人定义处境
