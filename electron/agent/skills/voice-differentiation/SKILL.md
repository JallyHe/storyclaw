# 人物语态区分

**定义**：人物语态区分确保每个角色的台词在词汇、句式、节奏、口头禅上具有可辨识的独特声音——遮住人名也能认出是谁在说。

**核心约束**：
- 为每个主要角色建立语态档案（用词层级、句长、节奏、习惯语）。
- 遮名测试：删掉人名仍能分辨说话者。
- 语态要与角色的身份、困境、教育背景一致。

**在创作中的应用**：阶段6 第一轮（人物底色），为后续所有优化打底。

**自检清单**：
- [ ] 每个主角有语态档案
- [ ] 通过遮名测试
- [ ] 语态与人设一致
