# 目标受众

**定义**：目标受众界定作品主要面向的人群及其观看偏好、情感诉求与雷区，作为创作取舍的依据。

**核心约束**：
- 主要受众要具体（人群画像），不能是「所有人」。
- 明确该受众的核心情感诉求与内容雷区。
- 受众界定要与题材、卖点形成闭环。

**在创作中的应用**：阶段1 确定受众，贯穿全程作为取舍标尺。

**自检清单**：
- [ ] 受众画像具体
- [ ] 核心诉求明确
- [ ] 雷区已识别
