此 skill 定义和阐述"目标受众"（Target Audience）——从故事内容推断出故事针对的是哪一类人群。

## 核心定义

**从故事本身的内容——人物设定、冲突类型、情感模式、世界观、节奏、语言风格——反向推断出这个故事"天然地为谁而写"。目标受众不是作者宣称的，是故事内容自己指向的。**

一个故事不可能为"所有人"而写。"所有人都会喜欢"是目标受众分析失败后的遮羞布。好的受众洞察是：承认这个故事会让某一群人如痴如狂，而另一群人完全无感——并且知道为什么。

## 推断维度

从以下维度逐一扫描故事材料，每个维度都会缩小目标受众的范围：

### 1. 主角的身份与年龄
主角的年龄阶段、职业状态、人生阶段——决定了"谁会在主角身上看到自己"。

- 主角是在校学生 → 核心受众大概率在同龄段
- 主角是初入职场的新人 → 核心受众处于或即将进入同一阶段
- 主角是中年危机中的父母 → 核心受众经历过或正在经历这一阶段

### 2. 核心冲突的类型
冲突的类型暗示了"谁会觉得这个冲突与自己有关"。

- 职业竞争/升迁冲突 → 职场人群
- 亲密关系中的信任/背叛 → 正在或曾经在亲密关系中的人
- 自我认同/成长困境 → 处于人生转型期的人
- 生存/安全威胁 → 更广泛的大众受众（普世恐惧）

### 3. 情感满足的模式
故事给观众提供的情感报酬是什么类型——这决定了"谁缺乏这种报酬"。

- 被看见、被理解的感觉 → 感到被忽视的人群
- 逆袭、翻盘的爽感 → 在现实中感到压抑或不公的人群
- 完美的爱情体验 → 在现实中缺乏理想亲密关系的人群
- 恐惧中的安全感重建 → 在现实中感到不安的人群

### 4. 世界观的门槛
故事的设定、术语、文化引用——决定了"谁不需要额外解释就能进入这个世界"。

- 高语境行业背景（如医疗、法律、金融）→ 对这些行业有了解或兴趣的人
- 特定亚文化圈层（如电竞、嘻哈、二次元）→ 该圈层的参与者或向往者
- 特定历史时期 → 对该时期有了解或情感连接的人

### 5. 节奏与复杂度
故事的叙事节奏和结构复杂度——决定了"谁的注意力模式与这个故事匹配"。

- 快节奏、高冲突密度 → 年轻的、习惯高信息流输入的人群
- 慢热、层层铺陈 → 有耐心、享受沉浸式叙事的人群

## 输出格式

对目标受众的描述不能只是一个笼统的 demographic 标签。必须包含以下层次：

**1. 核心受众**：[年龄范围 + 性别倾向（如有）+ 核心的共享特征（身份/处境/心理状态）]

**2. 为什么是这群人**：[从故事内容中提取至少 3 个具体元素，逐一说明它们为什么指向这类人群]

**3. 这群人会在什么场景下消费这个故事**：[通勤路上？睡前？周末沉浸？社媒碎片时间？]

**4. 不会喜欢这个故事的人群**：[明确说出哪类人会对这个故事无感或反感 + 原因]
