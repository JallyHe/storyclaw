此 skill 对已有故事材料进行结构化的架构提取与总结。不评价优劣，不诊断问题，不提出改进——只忠实还原和梳理材料中已有的内容。

## 核心原则

**忠实还原，不编造。** 每一项输出必须有材料中的依据。材料中不存在的——标注"暂缺"，不凭空填充。

---

## 输出一：故事结构总结

> 忠实还原材料中的情节结构骨架。不判断好坏，不指出缺陷，不比较优劣。

### 一、故事线总览

从材料中提取并简述：

- **主线**：[用材料中的情节概括，一句话]
- **角色成长线**（每条单独列出）：[角色名]：从 [材料中的起点状态] → [材料中经历了什么] → [材料中的终点状态/当前状态]
- **关键关系线**（每条单独列出）：[角色A] 与 [角色B]：[材料中关系的起点] → [材料中关系的变化节点] → [材料中关系的当前状态]

---

### 二、故事情境序列（C）—— 最详细部分

按时间顺序逐一描述每一个故事情境。这是整个输出中篇幅最长、细节最密的部分。

对每个 C，输出：

**C[N]：[情境名称]**

- **情境定义**：角色面临的不平衡状态是什么？用材料中的具体情节来描述。
- **起点**：由哪个具体事件触发？引用材料中的情节。
- **终点**：由哪个具体事件终结？引用材料中的情节。
- **赌注**：如果这个情境不被解决，材料中展示或暗示的代价是什么？
- **情境类型**：信息型 / 冲突型 / 目标型 / 存在型

**属 C[N] 的 D 情节**（进入此情境的情节）：

- 触发事件 + 角色做出的选择 + 进入情境时被确认的代价

**属 C[N] 的 A 情节序列**（此情境内的循环尝试）：

- **A1**：尝试 → 部分成功 → 复杂化 → 结果
- **A2**：同上
- …（按材料中实际出现的次数）

**属 C[N] 的 B 情节**（终结此情境的情节）：

- 终结事件 + 旧情境如何被改变 + 新情境的种子（开启的下一个 C 是什么）

> 如果某个 C 下的 ABD 情节在材料中不完整或无法识别——如实标注，不强行填充。

---

### 三、结构关系总图

用箭头图展示所有 C 之间的递进关系：

```
D1 → C1 [名称]
        ├── A1 → A2 → ...
        └── B1 → C2 [名称]
                    ├── A3 → A4 → ...
                    └── B2 → C3 [名称]
                                └── ...
```

## 输出二：人物小传

> 以每个主要角色为主体，从其 POV 梳理故事线，标注角色类型和角色情境。

对材料中每个主要角色，输出：

### [角色名]

**1. POV 故事线**

以该角色的视角重新叙述整个故事。沿着该角色的"欲望→行动→受阻→调整→新的行动"的脉络，重述一条完整的故事线。回答：**这个角色从头到尾在追逐什么？他在每个阶段的选择是什么？他的结局/当前状态是什么？**

**2. 角色功能类型**

从以下类型中识别（可以多重重叠），并引用材料中的具体内容作为依据：

- **代入型**：观众借其眼睛体验故事。依据：该角色在材料中的哪些具体设定或行为使观众能在其身上看到自己？
- **奖励型**：为代入型角色提供情感报酬（爱情/认可/友情/支持/付出）。依据：该角色在材料中的哪些具体互动提供了哪种类型的奖励？
- **陪伴型**：与代入型角色组队同行，提供陪伴和对话。依据：该角色在材料中的哪些具体场景中完成了"让主角不孤单"的功能？
- **情节发动型**：因其存在和行动不断推动情节运转（对手/闯祸队友/攻略对象/催化者）。依据：该角色在材料中具体发动了哪些情节？

**3. 角色情境**

- **内置型摩擦**：该角色的什么特质使他/她与周围环境始终存在某种张力？引用材料中的具体表现。
- **目标型摩擦**（如有）：该角色有什么始终想要但始终未得到的目标？引用材料中的具体表现。
- **在材料中的贯穿性**：这个摩擦在材料中是否从头到尾都存在？

## 输出三：故事类型与受众分析

> 从题材和受众角度梳理故事。所有分析基于材料中的具体内容。

### 一、题材定位

- **完整题材描述**：[为 XX 人群打造的 XX 题材，聚焦 XX]
- **传统题材分类**：[主要类型] + [子类型] + [辅助类型（如有）]
- **题材引力中心**：这个故事用什么元素驱动观众追看？（谜题 / 关系 / 成长 / 世界观 / 情绪）
- **同类设定/角度**：该题材下，这个故事采用的设定和角度。引用材料中的具体内容。

### 二、目标受众

- **核心受众**：[年龄范围 + 性别倾向（如有）+ 核心共享特征]
- **推断依据**：从材料中提取至少 2 个具体元素（人设/情节模式/情感模式/世界观），逐一说明它们指向哪类人群
- **与这个故事可能不太匹配的人群**：基于材料中的内容特征，哪类人群可能对这个故事不太感兴趣 + 原因

### 三、观众洞察

- **核心情感满足**：[一句话] 这个故事给目标受众提供的最核心的情感满足是什么？
- **对应的心理需求**：被爱 / 被看见 / 掌控 / 逃避 / 成长……（选最核心的 1-2 个）
- **满足方式**：故事中反复出现的叙事模式是什么——每次激活时观众获得什么具体情绪？引用材料中的情节。
- **受众现实处境的关联**：目标受众在现实中可能正在经历什么，使得这种情感满足对他们尤为重要？

### 四、社会洞察

- **与当下社会观念/情绪的同频点**：[一句话]
- **故事中的具体体现**：从材料中提取至少 1 个反映当下观念的具体元素
- **时代背景关联**：这个故事中的观念或情绪，与当下社会正在经历的变迁有什么关联？

---

## 输出规范

- 输出一：只忠实还原材料中的情节结构，不做任何优劣判断。最详细的部分是 C 及其下属 ABD 情节的描述。
- 输出二：每个角色的 POV 故事线必须是从该角色的眼睛看出去的故事，不是第三方概述其行为列表。
- 输出三：所有分析基于材料中的具体内容，可以在此基础上进行受众层面的推演。
- 所有输出中，材料不存在的元素标注"暂缺"并简述原因，不编造。
- 全文不使用"好/坏""强/弱""优/劣""成功/失败""应该/不应该"等评判性措辞。
