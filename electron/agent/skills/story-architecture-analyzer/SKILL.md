# 故事架构分析

**定义**：故事架构分析是对输入素材进行系统解构，提炼其核心主题、主要张力、人物格局与结构骨架，为改编策略提供依据。

**核心约束**：
- 先抓主题与核心张力，再梳理人物与结构。
- 区分原著的「不可动摇的内核」与「可改编的外壳」。
- 分析结论要可直接服务于 Logline 与改编边界的确定。

**在创作中的应用**：阶段1 第一步，作为核心策略文档的事实基础。

**自检清单**：
- [ ] 核心主题明确
- [ ] 主要张力提炼到位
- [ ] 人物/结构骨架清晰
- [ ] 区分内核与外壳
