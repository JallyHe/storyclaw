# 制作成本评估

**定义**：制作成本评估从制片角度识别剧本中的高成本要素（大场面、特效、外景、群演、特殊道具/年代），估算相对成本并提示优化空间。

**核心约束**：
- 标记高成本桥段，说明成本来源。
- 在不伤害核心表达的前提下，给出降本替代方案。
- 区分"贵但必要"与"贵且可省"。

**在创作中的应用**：制片可行性专家用来出具成本提示与优化建议。

**自检清单**：
- [ ] 标记高成本桥段
- [ ] 说明成本来源
- [ ] 给降本替代
- [ ] 区分必要/可省
