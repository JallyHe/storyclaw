# 冲突即戏剧

**定义**：冲突即戏剧：有戏的对白本质是两个意志的对抗。每段对话都应有潜在的拉锯、博弈或不对等，而非平和的信息传递。

**核心约束**：
- 每段对白找出对抗双方的相反目标。
- 把「报信息」的台词改写成「争夺什么」的台词。
- 冲突可显可隐，但不能没有。

**在创作中的应用**：阶段6 R4，把平淡对话改造为有张力的博弈。

**自检清单**：
- [ ] 每段对白有对抗
- [ ] 信息交换已戏剧化
- [ ] 冲突贯穿对话
