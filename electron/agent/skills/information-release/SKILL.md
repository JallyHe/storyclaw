# 信息释放

**定义**：信息释放控制台词层面信息披露的时机与剂量——配合 hole-before-fill 与信息位，决定每句话透露多少、留多少。

**核心约束**：
- 关键信息分批释放，制造悬念与期待。
- 不一次性倒空，也不无故拖延到观众失去耐心。
- 释放节奏与情境压力曲线匹配。

**在创作中的应用**：阶段6 R5，精修台词层面的信息节奏。

**自检清单**：
- [ ] 关键信息分批给出
- [ ] 节奏不拖沓
- [ ] 与压力曲线匹配
