# 情境铺垫

**定义**：情境铺垫（D）是把故事从旧状态推入新 C 的过渡——通常是一个触发事件，迅速建立新情境的张力前提。

**核心约束**：
- D 要快，体量压缩到最小，不要在铺垫上消耗观众耐心。
- D 结束时新 C 的核心张力必须已经成立。
- D 与前一个 B 可以重合（一个事件既终结旧 C 又触发新 C）。

**在创作中的应用**：阶段2 Step 2B 为每个 C 设计 D；阶段3 在梗概中标注 D 节点。

**自检清单**：
- [ ] 触发事件清晰
- [ ] 体量足够小
- [ ] 结束时新 C 已成立
