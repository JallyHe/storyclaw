# 人物关系图

**定义**：人物关系图梳理主要角色之间的关系（亲疏、敌友、利益、情感）与彼此张力，揭示关系如何随情境演变。

**核心约束**：
- 每对关键关系标注初始状态、核心张力与变化方向。
- 关系网要能制造冲突，避免一团和气。
- 关系变化必须由情节事件驱动，而非凭空转变。

**在创作中的应用**：人物设定专家用来产出关系图与关系弧。

**自检清单**：
- [ ] 关键关系标注张力
- [ ] 关系能制造冲突
- [ ] 变化由事件驱动
- [ ] 覆盖主要角色
