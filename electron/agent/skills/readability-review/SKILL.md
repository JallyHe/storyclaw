# 可读性审查

**定义**：可读性审查关注剧本文本层面的清晰度——场次标注、动作描述、对白格式是否规范，是否好读、无歧义，便于主创与制作团队理解。

**核心约束**：
- 动作描述简洁可视，避免文学性堆砌与内心独白。
- 场次/转场/人物/对白格式统一规范。
- 消除歧义表达，确保任何读者理解一致。

**在创作中的应用**：责编专家用来做文本层面的规范化审查。

**自检清单**：
- [ ] 动作描述简洁可视
- [ ] 格式统一
- [ ] 无歧义
- [ ] 便于制作理解
