# 情境为王

**定义**：情境为王：每一场戏都必须要么在构建当前情境的压力，要么在推动情境的解决；两者都不做的戏应删掉或压缩。

**核心约束**：
- 逐场问：这场戏服务于哪个 C？是加压还是解压？
- 答不上来的戏 → 删除或压缩合并。
- 再「好看」的桥段，若不服务于情境，也要修剪。

**在创作中的应用**：阶段2-5 全程作为筛选器；阶段4 分场时每一场都要标注其情境功能。

**自检清单**：
- [ ] 每场戏都对应一个 C
- [ ] 每场戏明确加压或解压
- [ ] 无功能场次已删除/压缩
