此 skill 定义和阐述故事架构原则：**通过重复与对比强化角色**（Reinforce Character Through Repetition and Contrast）。

## 核心原则

**角色的核心特质不是通过一次性的高光时刻建立的——而是通过"重复行为"让观众确认"这就是他"，再通过"情境对比"让观众看到"他在不同条件下还是他"。重复建立辨识度，对比建立深度。**

一个角色在一种情境下的行为是"信息"。在三种完全不同的情境下展现出同样的行为模式——这个模式就从"信息"升格为"人格"。观众不需要被告知"他是一个固执的人"——他们在三场完全不同的戏里看到他三次用三种不同的方式表现了同一个固执，他们已经知道了。

## 重复：建立角色的行为纹路

重复不是"做一样的事"——是"在不同的条件下暴露同一个底层逻辑"。

### 重复的方式

- **语言习惯的重复**：同一类句式、同一种措辞偏好、同一个口头禅——但每次使用的语境不同。
- **决策模式的重复**：面对不同类型的困境时——角色每次都做出同一类选择。观众开始能预测他的选择——这不是"可预测=无聊"，而是"可预测=这个角色是真实的"。
- **反应方式的重复**：面对赞扬、面对攻击、面对沉默——角色的反应方式是一致的。他不一定每次都说一样的话，但他每次都用同一种策略。

### 重复的节奏

- 建立：第一次出现——观众注意到，但不一定记住。
- 确认：第二次出现——观众意识到"这和他上次做的事一样"。
- 加固：第三次出现——观众已经不需要被说服。这个特质就是这个角色。

## 对比：在不同情境中暴露同一底色的不同面

对比不是让角色"改变"——是把他放在不同的情境中，让他的同一底色在不同条件下折射出不同的光谱。

- **高压情境 vs 低压情境**：同一个角色在危机中冷静、在日常中焦躁——不是矛盾，是"他的底色在高压下表现为冷静，在低压下反而不知道该怎么办"。
- **对强者 vs 对弱者**：同一个角色对上级毕恭毕敬、对下级冷漠——不是伪善，是"他的底色中有一个他自己都不知道的等级意识"。
- **有人在场 vs 独处**：同一个角色在人前乐观、在独处时沉默——不是伪装，是"乐观是他的社交工具，沉默是他的真实温度"。

对比不是让角色矛盾——对比让角色的底色更丰富。同一个底色，不同角度的光照——看到的不同颜色都是同一块石头的不同切面。

## 强化方向（Reinforcement Direction）

从 story-restructurer 的 Phase 2（Character Analysis）中提取：在分析每个角色时，确定——

**这个角色的核心特质，在故事的进程中，应该被"加固"还是被"复杂化"？**

- **加固**：观众第一次见到的那个特质——在故事的全程中被不断确认和加深。这个角色不会"改变"——他会"越来越像自己"。适合：系列主角、喜剧角色、符号型角色。
- **复杂化**：观众第一次见到的那个特质——在故事推进中被一层层揭开，露出下面更不为人知的层面。每一层都是对上一层解释的重新扩充。适合：成长型主角、反转角色、悲剧主角。

## 与同组原则的关系

- 与 [[situation-is-king]] 配合 —— 角色在不同情境中的重复表现，必须始终在服务故事情境的引力中心
- 与 [[reverse-thinking]] 配合 —— 反向思考要求好角色看起来像压力来源——对比手法可以用来在同一个角色上展示"表面压力"和"真实善意"两个面
- 与 [[hard-choices]] 配合 —— 角色的决策模式通过重复的选择被观众识别，而最难的抉择是这一模式的最高考场
