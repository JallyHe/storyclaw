# 艰难抉择

**定义**：艰难抉择是逼迫角色在两个都有重大代价的选项间做选择的关键设计——通过抉择揭示角色的真正价值排序。

**核心约束**：
- 两个选项都要有真实、重大的代价，不能有明显「正确答案」。
- 抉择要从角色的永久困境中自然生长出来。
- 抉择的结果应改变角色处境或关系，推动情境。

**在创作中的应用**：阶段3 标注关键抉择点；阶段4-6 充分展开抉择的张力与后果。

**自检清单**：
- [ ] 两选项均有重大代价
- [ ] 抉择源自永久困境
- [ ] 结果推动情境
