此 skill 定义和阐述"题材"（Genre Classification）——将传统题材分类与目标受众描述结合，形成对故事类型的完整定义。

## 核心原则

**题材描述不能只是传统的类型标签。"悬疑""爱情""职场"——这些词在内容爆炸的时代已经失去了辨识度。一个有意义的题材描述，必须在传统分类之前加上"目标观众描述"——回答"这是谁的故事，讲给谁听"——然后才是类型的锚定。**

形式：**"为 [目标观众] 打造的 [传统题材]，聚焦 [核心情感/冲突]"**

## 传统题材分类框架

### 叙事驱动型题材（以"发生了什么"为引力中心）

**悬疑 / 推理**
- 核心引力：一个需要被解开的谜——谁是凶手？真相是什么？
- 变体：心理悬疑（焦点不在"谁做的"，在"为什么做"）、法庭推理、硬汉侦探、本格推理

**惊悚 / 恐怖**
- 核心引力：一个迫在眉睫的威胁——谁会死？能不能逃出去？
- 变体：心理惊悚（威胁来自人心）、超自然恐怖、生存恐怖、科技惊悚

**犯罪 / 黑帮**
- 核心引力：法律与秩序的边界——他能逃脱吗？正义会降临吗？
- 变体：黑帮史诗、劫案、警察办案、法医侦查

### 情感驱动型题材（以"角色之间的关系"为引力中心）

**爱情**
- 核心引力：两个人能否跨越阻碍在一起？
- 变体：甜宠、虐恋、先婚后爱、破镜重圆、暗恋、禁忌之恋

**家庭 / 伦理**
- 核心引力：家庭关系中的裂痕能否被修复？
- 变体：代际冲突、婚姻危机、育儿、养老

**友情 / 群像**
- 核心引力：一群人的羁绊在考验中能否维持？
- 变体：死党组、成长群像、团队逆袭

### 成长驱动型题材（以"角色变成什么样的人"为引力中心）

**青春 / 成长**
- 核心引力：主角在青春期/转型期中如何找到自己？
- 变体：校园成长、残酷青春、治愈成长

**职场**
- 核心引力：主角在职业世界中如何获得成功/意义？
- 变体：行业剧（医疗/法律/金融）、创业、底层打拼

### 世界观驱动型题材（以"这个世界的规则是什么"为引力中心）

**科幻**
- 核心引力：一个新科技/新规则如何改变了"人"的定义？
- 变体：硬科幻、赛博朋克、太空史诗、反乌托邦

**奇幻**
- 核心引力：在一个魔法/超自然力量存在的世界中，角色的命运如何不同？
- 变体：史诗奇幻、都市奇幻、修仙、异能

**历史**
- 核心引力：在某个特定历史时期，角色的命运如何被时代裹挟？
- 变体：历史正剧、架空历史、穿越、宫斗

### 情绪驱动型题材（以"观众想获得什么感受"为引力中心）

**喜剧**
- 核心引力：观众想笑——想从日常的沉重中获得释放。
- 变体：情景喜剧、黑色喜剧、浪漫喜剧、讽刺喜剧

**动作 / 冒险**
- 核心引力：观众想体验刺激——极致的身体冒险和视觉冲击。
- 变体：功夫、枪战、夺宝、灾难

## 合并格式：受众 + 题材

题材描述的核心格式：

> **"为 [目标受众特征] 打造的 [传统题材分类]，聚焦 [这个故事独特的情感/冲突锚点]"**

### 示例

- 弱："这是一个悬疑题材。"
- 强："为 25-35 岁都市女性的'智性恋'审美打造的 心理悬疑，聚焦一个用心理学破案的女心理师，在追踪真相的过程中发现——她自己才是被观察的对象。"

- 弱："这是一个爱情故事。"
- 强："为 20-28 岁在现实中经历了'恋爱降级'的年轻女性打造的 甜宠爱情，聚焦一个不相信爱情的女主角，在一个完全尊重她边界感的高情感智能男性面前——如何被'治愈'而非'拯救'。"

- 弱："这是一个逆袭爽文。"
- 强："为 22-30 岁在北上广深'卷不动又躺不平'的年轻人打造的 职场逆袭爽剧，聚焦一个被全公司最看不起的底层员工，用所有人都觉得'没用'的冷门技能——在每一集打脸一个不同的上位者。"

## 多重题材标记

一个故事可以同时属于两个或多个题材分类——但不能是"什么都有"。多重题材必须是互补而非冲突的：

- ✓ 心理悬疑 + 爱情 = 两个人在追查真相的过程中建立信任和感情
- ✓ 职场 + 喜剧 = 在办公室政治的荒谬中获得喜剧释放
- × 悬疑 + 治愈 = 两个类型的核心情感体验互相矛盾（恐惧 vs 安心）

## 输出格式

**1. 完整题材描述**：[受众 + 题材 + 核心焦点] 一句话

**2. 传统题材分类**：[主要类型] + [子类型] + [辅助类型（如有）]

**3. 题材引力中心**：这个故事用 [什么元素] 驱动观众持续追看？[谜题 / 关系 / 成长 / 世界观 / 情绪]

**4. 同类题材中的差异点**：同为 [X 题材]，这个故事与其他同类题材作品最不一样的设定/角度是什么？
