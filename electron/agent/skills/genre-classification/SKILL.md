# 题材定位

**定义**：题材定位明确作品归属的类型（及其混合），并据此调用对应的类型惯例与观众期待。

**核心约束**：
- 主类型 + 次类型的组合要清晰。
- 尊重核心类型惯例，差异化体现在执行而非违背惯例。
- 题材定位要与目标受众一致。

**在创作中的应用**：阶段1 确定题材，指导后续张力类型与卖点设计。

**自检清单**：
- [ ] 主/次类型清晰
- [ ] 类型惯例已识别
- [ ] 与受众一致
