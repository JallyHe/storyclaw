# 因果链校验

**定义**：因果链校验逐一检查事件之间的因果关系是否成立——每个关键转折是否有充分的前因，有无依赖巧合或角色降智推动剧情。

**核心约束**：
- 每个关键事件都要能回答"为什么会发生"，前因充分。
- 禁止靠巧合或角色突然变蠢/变聪明推动剧情。
- 断裂处给出补因建议。

**在创作中的应用**：逻辑校对专家用来排查因果断裂与降智。

**自检清单**：
- [ ] 关键事件前因充分
- [ ] 不靠巧合推进
- [ ] 无角色降智
- [ ] 断裂处给补因
