# Phase 2 技法详解（Techniques Detail）

本文件是 `plot-to-screenplay` 技能 Phase 2 的配套深读材料。SKILL.md 给出每个技法的**定义**，这里给出**展开说明 + 可操作步骤 + 正反例**。所有示例围绕同一场戏递进，演示「技法是一层层叠加到同一个正确底稿上的」——这正是 Phase 2 的工作方式。

> 贯穿铁律：**技法改变信息「怎么传」，绝不改变信息「传不传」。** 每做完一层都要回头确认：原底稿里该到达观众的信息，是否仍然到达。

---

## 递进示例的底稿（Phase 1 产物）

场景：深夜，公司财务室。老陈（财务总监，年长、稳重）发现账目有问题，怀疑是下属小林做的。小林（入职两年、表面老实）来还钥匙。Phase 1 底稿是「逻辑正确但平」的版本：

```
【场景 7：财务室 —— 夜】
人物表：老陈、小林
△ 老陈坐在电脑前看账目。小林敲门进来。
小林
陈总，钥匙还您。
老陈
账目我看了，三月那笔二十万对不上。是你做的吗？
小林
是我做的，但我没动钱，可能是系统错了。
老陈
我相信你。你先回去吧。
```

信息清单（必须全程保住）：① 账目有 20 万对不上；② 老陈怀疑小林；③ 小林承认是自己经手、否认贪污；④ 老陈嘴上信任、实则未必。下面每一层都在不丢这四条的前提下加戏。

---

## Layer 1：人物底色

### 技法 1 · 人物声音差异化（Voice Differentiation）
**做什么**：在加任何戏之前，先让「谁在说话」从台词本身可辨。遮住人名仍能认出说话人，这一层才算过。
**抓手**：词汇层级（老陈用书面、克制词；小林用口语、回避词）、句子长度（老陈短而定，小林长而碎）、语言习惯（老陈爱用反问压人，小林爱加「可能/应该/其实」）。
**正例**：

```
老陈
三月。二十万。对不上。
小林
那个……其实我也纳闷，按理说不应该啊，可能是系统那边……
```

老陈三个短句像三记敲钉；小林一连串软词暴露心虚。**没有这层，后面所有技法都会变成「作者借角色之口说话」。**

## Layer 2：核心三支柱

### 技法 2 · 潜台词优先（Subtext First）
**做什么**：人物几乎不直说真意，用暗示、回避、转移话题、反话、动作来编码。观众感到「说出口的」与「想说的」之间有落差。
**抓手**：把直白台词翻译成「不直说但能被读出」的版本；**编码不是裁剪**——信息仍要可被接收。
**反例（把信息删没了，错误）**：

```
老陈
（沉默，没问账目）你先回去吧。
```

→ 观众收不到「老陈在怀疑」，信息丢失，这是事故。
**正例（编码但信息仍在）**：

```
老陈
（没抬头）钥匙搁着吧。……三月挺忙的吧？一个人盯二十多笔。
小林
还……还好。
老陈
辛苦。
```

老陈没说「我怀疑你贪了二十万」，但「三月」「二十多笔」「辛苦」三处暗钉，把怀疑编码进了关心里。观众依然收全了①②③④。

### 技法 3 · 元评论（Meta-Commentary）
**做什么**：人物不只是「说」，还不断「对自己正在说的话、对当下处境」作评论。元评论是潜台词最常见的载体——它提供一层安全外壳，把真实情绪藏在「评论」后面。
**形式**：自嘲、先点评对方的话再回答、给当前局面贴标签、表面夸赞暗含锋芒、意味深长的沉默。
**正例**：

```
小林
陈总今天问得这么细，搞得我像来自首的。
老陈
（笑）你看你，我就问个账，你先把「自首」俩字说出来了。
```

小林用「自首」自嘲，是给紧张贴标签的安全壳；老陈顺着这个词反将一军——真实交锋藏在玩笑的评论层下。

### 技法 4 · 冲突即戏剧（Conflict Is Drama）
**做什么**：每一组对话都要含对抗。若两人全程一致，要么戏坏了，要么这「一致」掩盖着更深的冲突。
**四型**：立场冲突（要的东西不同）、信息冲突（知道的不同）、地位冲突（争夺控制权）、内在冲突（角色自我矛盾外泄）。本场是**信息冲突 + 地位冲突**：老陈知道「有人动手脚」、想拿到承认；小林知道实情、想全身而退；同时两人都在争「这场谈话由谁定调」。
**抓手**：给每句台词标一个「此刻想从对方那里得到什么」，若标不出，这句多半是信息播报，需重写或删。

## Layer 3：信息策略

### 技法 5 · 信息层层递进（Information Release）
**做什么**：信息不一次倒完，像剥洋葱。每句只释放此刻观众需要的那一层，控制「观众所知」与「各角色所知」之间的差。
**做法**：把信息清单排序，决定**释放顺序**与**释放人**。本场顺序设计：先放「账目异常」（环境/老陈），再放「老陈早有怀疑对象」（潜台词），再放「小林经手」（小林自曝），最后**保留**「老陈信不信」（不在本场解，留作钩子）。

### 技法 6 · 问答角色翻转（Q&A Role Reversal）
**做什么**：打破「一问一答」。让本该提问的人自问自答，让本该回答的人反问。让**性格**而非剧情的信息需求决定信息怎么给出。
**反例（被剧情绑架的问答）**：

```
老陈
是你做的吗？
小林
是我，但我没动钱。
```

**正例（翻转）**：

```
老陈
（不问，自答）我猜你要说，是你做的，但你没动钱。
小林
（怔住）……您都想好我怎么答了。
老陈
我不想好，怎么知道你是不是在背台词。
```

老陈把小林的回答提前替他说了——提问者自答，反而把压力顶到小林脸上；信息（③小林经手、否认贪污）仍然送达，但路径由老陈的强势性格决定。

## Layer 4：戏剧手段

### 技法 7 · 回声呼应（Callback / Echo）
**做什么**：让后文引用前文的台词或动作产生叠加效应。三型：引对方的话来反驳、重复自己的话来强调、第三方重复以达讽刺或共情。
**前置**：需要前面技法已埋下可回收的「料」。本场回收技法 3 的「自首」：

```
老陈
（小林走到门口）小林。
小林
（回头）
老陈
刚才那个词——自首。回去想想，是不是你自己先慌的。
```

「自首」在结尾回声，把一句玩笑变成压在小林心上的钉子。

### 技法 8 · 打脸 / 言行不一（Face-Slap）
**做什么**：制造「刚说完就被现实打脸」的反差——人物前一刻的断言被动作或情境当场拆穿。
**前置**：需要技法 1 已立起稳定的行为模式可供「自拆」。

```
小林
我问心无愧，账您随便查。
△ 老陈把屏幕转过来，正停在小林三月的操作日志上。小林的手不自觉地收了回去。
小林
（声音低下去）……这个我可以解释。
```

「随便查」与「手收回来」当场对撞，无需旁白，观众自己读出心虚。

### 技法 9 · 紧张断点（Tension Break）
**做什么**：在张力最高处，插入一个与对话完全无关、极短的打断（外部闯入 / 物件意外 / 日常动作），把情绪悬置以增压。**限制：一场不超过两次，每次不超过三行。**

```
小林
（声音低下去）……这个我可以解释。
△ 走廊尽头，保洁的拖把桶哐当一声倒了。两人都顿了一下。
老陈
（视线没离开小林）解释。
```

拖把桶的意外把「解释」二字悬了半拍，张力不泄反涨。

## Layer 5：润色收束

### 技法 10 · 动作催化对白（Action Catalyzing Dialogue）
**做什么**：把动作描写放在关键台词**之前**（蓄能）、**之后**（释放）、或**之中**（节奏停顿）。动作与对白互相催化，单独都不完整。

```
△ 老陈摘下眼镜，慢慢擦。（蓄能）
老陈
我做财务二十年，最怕的不是有人偷钱。
△ 他把眼镜戴回去，看住小林。（之中停顿）
老陈
是偷了钱，还看着我的眼睛说没有。（释放）
```

### 技法 11 · 节奏控制（Rhythm Control）
**做什么**：用语速与停顿匹配场景情绪。紧则收（短句、少停顿），缓则放（长句、多留白）。一场戏的对白节奏是它的心跳；若每场都同一速度，观众就停止感受。
**本场设计**：开场缓（老陈擦眼镜、闲笔铺垫）→ 中段提速（信息冲突、短句对刺）→ 打脸处最紧（半句、被打断）→ 结尾骤缓（「回去想想」留白收束）。

### 技法 12 · 删减冗余（Trim Redundancy）
**做什么**：能删则删——寒暄、重复、口水。**能用动作表达的，就不用台词。**
**铁律：删的是冗余的「表达」，不是信息。** 删前先问：这句承载什么信息？别处是否已有？删后观众是否仍掌握理解后文所需的一切？**先跑信息丢失检查，再跑节奏检查**——丢信息是事故，丢节奏可补救。
**示例**：把开场的

```
小林
陈总您还没下班啊，这么晚还在忙，要不要我帮您倒杯水？
```

压成

```
△ 小林倒了杯水推过去。老陈没碰。
```

寒暄删光，「小林想示好、老陈不接」的信息改由动作承载，更紧也更冷。

## 收尾自检（Phase 2 完成标志）
逐条确认，全部为「是」方算完成：
- [ ] 遮住人名仍能分辨每个角色（技法 1）
- [ ] 关键真意有潜台词承载，且信息**零丢失**（技法 2、12 铁律）
- [ ] 每组对话都含可标注的对抗（技法 4）
- [ ] 信息按设计的顺序与释放人逐层给出（技法 5、6）
- [ ] 至少一处回声/打脸/断点服务于本场高潮（技法 7-9）
- [ ] 动作与对白互相催化、节奏随情绪起伏（技法 10、11）
- [ ] 与底稿信息清单逐条比对，四条信息全部仍然到达观众
