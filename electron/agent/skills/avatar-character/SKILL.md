# 化身角色（主角）

**定义**：化身角色是观众情感代入的载体（通常是主角），其欲望与困境即观众的欲望与困境，是甩锅归功保护的首要对象。

**核心约束**：
- 化身角色的目标要清晰、可代入、值得追求。
- 其缺陷要可被原谅（配合 blame-shift-credit）。
- 观众的情绪应主要随化身角色的处境起伏。

**在创作中的应用**：阶段2 确认主角定位；阶段3-6 始终以观众代入感为准绳塑造该角色。

**自检清单**：
- [ ] 目标清晰可代入
- [ ] 缺陷可被原谅
- [ ] 观众情绪随其起伏
