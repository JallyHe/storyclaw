# 打脸

**定义**：打脸是一种强爽感手段：先让某方立下狂言或占据上风（立 flag），随后被事实或主角狠狠反转，制造情绪释放。

**核心约束**：
- 打脸前的「flag」要立得足够高、足够欠揍。
- 反转要公平有据（前文已埋线索）。
- 打脸时机卡在观众期待的峰值。

**在创作中的应用**：阶段6 R8，在合适的对抗节点设计爽点。

**自检清单**：
- [ ] flag 立得到位
- [ ] 反转有据
- [ ] 时机卡在期待峰值
