# 元评论（自我点评）

**定义**：元评论是让角色对自身处境、套路或观众预期做出带有自觉性的评论，以制造反讽、共鸣或幽默。

**核心约束**：
- 元评论要克制，过度会出戏、显得油滑。
- 必须服务于情境或人物，不是为了抖机灵。
- 与作品整体调性一致。

**在创作中的应用**：阶段6 R3，在合适处点缀以提升机锋。

**自检清单**：
- [ ] 使用克制
- [ ] 服务情境/人物
- [ ] 调性一致
