# 可拍摄性

**定义**：可拍摄性评估剧本在实拍层面的可行性——场景能否搭建/取景、动作能否安全完成、技术能否实现，识别难落地的桥段。

**核心约束**：
- 标记实拍困难点（高危动作、难取景、复杂调度等）。
- 给出可执行的替代拍法或改写建议。
- 兼顾安全与周期，不空谈艺术效果。

**在创作中的应用**：制片可行性专家用来出具可拍摄性评估。

**自检清单**：
- [ ] 标记实拍困难
- [ ] 给替代拍法
- [ ] 兼顾安全与周期
- [ ] 识别技术障碍
