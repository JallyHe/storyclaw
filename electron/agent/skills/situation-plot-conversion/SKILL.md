此 skill 定义和阐述"情境与情节的转换"（Situation-Plot Conversion）——故事结构体系中关于如何在情节和情境之间进行升降格操作的结构技巧。

## 核心原理

**情节和情境不是固定的分类——它们可以在结构层面互相转换。一个好的情节可以被"升格"为情境以延长其张力；一个弱的情境可以被"降格"为情节以加快节奏、释放篇幅。**

这种转换的核心判断标准只有一个：**当前的张力和赌注是否值得观众在这个状态中停留更长时间。** 如果值得——升格。如果不值得——降格。

---

## 操作一：情节升格为情境（Plot → C）

### 适用条件

当前的情节单元具备以下特征时，考虑将其升格为情境：
- 张力极强——观众在这一刻的追问力是最强的
- 赌注明确且具体——观众清楚地知道"如果这一步失败意味着什么"
- 角色在这一刻的选择具有不可逆性——一旦做出选择就无法回头
- 这个情节触及了故事的核心情感或核心冲突

### 操作方式

将一个本来的 A 情节或 D 情节升格为一个完整的 C——在它内部嵌入子 A 情节（sub-A-plots），让它从一个"被快速经过的事件"变成一个"观众在其中停留一段时间的状态"。

**结构变化**：

```
升格前：
  D → C1 → A1 → A2 → B1 → C2  （A2 是其中张力最强的A情节，但一闪而过）

升格后：
  D → C1 → A1 → B1 → C2
                  其中 C2 由原来的 A2 升格而来：
                  D2 → C2 → A2a → A2b → A2c → B2 → C3
```

**原始 A2 的内容**（升格前）：主角必须在五分钟内拆掉炸弹——这是一个紧张的情节，但它在故事中只占一场戏的篇幅。

**升格为 C2 后**：将"拆弹"从一个动作延展为一个情境——C2 就是"拆弹情境"。内部嵌入子 A 情节：
- A2a：主角尝试剪红线 → 倒计时暂停了一秒 → 然后以更快的速度下降（复杂化）→ 处境：倒数时间更短了
- A2b：主角尝试用另一个工具 → 工具断裂（复杂化）→ 处境：没有了备用方案
- A2c：主角在最后几秒做出了判断 → 剪对了 → B2 结束 C2

原本"拆弹"是一个场景——现在"拆弹"是整个 C2，观众在其中停留了更长时间，张力被保持、被递增。

### 升格后的效果

- **张力的延长**：观众在原本"一闪而过"的紧张中停留了更久——追问力没有减弱，反而因为子 A 情节的层层递进而被增强
- **赌注的放大**：每个子 A 情节都在重新确认"失败的代价是什么"——C2 的赌注在 A2a、A2b、A2c 中不断被观众重新感知
- **角色的深度**：角色在延长的压力中做出的每一个微小选择——都暴露出更多的性格层面

### 升格的判断标准

在决定是否升格之前，追问：
1. 这个情节的张力是否强到观众愿意在其中停留更长时间？
2. 这个情节内部是否包含足够的"可分解的动作"——可以拆出至少两个子 A 情节？
3. 角色在这个过程中面临的选择是否有足够的层次——不是一个"剪还是不剪"，而是"先试什么，失败后换什么"？

如果三个答案都是"是"——升格。

## 操作二：情境降格为情节（C → Plot）

### 适用条件

当前的故事情境具备以下特征时，考虑将其降格：
- 张力不足——观众对这个状态的追问力弱
- 赌注模糊——观众不清楚"如果这个情境不被解决会怎样"
- 角色在这个情境中的行动缺乏不可逆性——随时可以退出
- 这个情境内部缺乏足够支撑多个 A 情节的"可分解的动作"

### 操作方式

将一个本来的 C 压缩为一个 D 情节或 B 情节——删减其内部的 A 情节序列，只保留"状态改变"的功能，让它从一个"被充分展开的情境"变成一个"快速经过的转折"。

**结构变化**：

```
降格前：
  C1 → B1 → D2 → C2 → A3 → A4 → A5 → B2 → C3

降格后：
  C1 → B1 → D2（原 C2 降格为 D2）→ C3
         其中 D2 直接触发 C3，
         原 C2 中的 A3、A4、A5 全部删除或压缩为 D2 中的一个简述
```

**原始 C2 的内容**（降格前）：主角需要攒够一笔钱才能采取行动——这个C持续了三集，内部有三个A情节（尝试赚钱的各种方式）。

**降格为 D2 后**：将"攒钱"从三集的情境压缩为一个快速过场——"她用三个月攒够了钱"——然后直接进入下一个更有张力的 C。观众不需要看每一笔钱是怎么赚到的——观众只需要知道"钱够了，她现在可以行动了"。

### 降格后的效果

- **节奏的加快**：删除了观众不太关心的"过程"，直接进入观众急于看到的"结果"
- **焦点的集中**：将篇幅释放给更有张力的 C，故事的重心更清晰
- **避免泄力**：弱情境如果在故事中占据太多篇幅，观众会在其中流失注意力——降格可以止损

### 降格为 D 还是 B？

- **降格为 D**：如果这个状态改变了，而且改变后的新状态开启了一个新的 C——保留它作为 D 情节。因为 D 情节的功能是"将故事从旧状态推入新 C"
- **降格为 B**：如果这个状态的结束本身就是对前一个 C 的终结——保留它作为 B 情节。因为 B 情节的功能是"结束旧 C，释放压力"
- **注意**：状态改变了，所以不太容易处理成 A 情节——A 情节的特征是"尝试解决但最终不改变情境"。状态既然已经变了——就不再属于 A 情节的范畴

### 降格的判断标准

在决定是否降格之前，追问：
1. 观众是否在追问"这个情境什么时候结束"而不是"这个情境内部会发生什么"？
2. 这个 C 的赌注是否可以用一句话概括——而概括之后发现"好像也没什么大不了"？
3. 删除这个 C 的 A 情节序列后——故事的核心情感线是否受到实质性影响？

如果第一个问题的答案是"是"，后两个是"否"——降格。

## 两种操作的关系

升格和降格是同一个结构思维的两种方向：

- **升格**：发现了好东西 → 不要让它一闪而过 → 把它变成观众可以停留的状态
- **降格**：发现了平淡的东西 → 不要让它占据篇幅 → 把它压缩成一个快速经过的转折

一个好的故事结构不是"每个 C 都一样长"——而是"张力越强的 C，篇幅越长；张力越弱的 C，篇幅越短"。升格和降格就是调节这种篇幅分布的工具。

---

## 常见错误

### × 对一个没有足够张力的情节强行升格
如果没有足够的内在层次支撑，升格后的 C 会出现"子 A 情节重复、压力不递增"的问题。升格的前提是原情节本身就有被延展的潜力。

### × 对一个关键的 C 因"太难写"而降格
降格不是为了逃避——是因为这个 C 真的不需要那么长。如果是因为"不知道怎么写"而降格——这是一个写作问题，不是一个结构问题。

### × 降格后丢失了重要的情感节拍
降格压缩的是情节，不是情感。如果一个 C 虽然张力不足但承载了重要的角色关系变化——降格时需要把关系变化节拍保留在 D 或 B 中，而不是连同 A 情节一起删除。
