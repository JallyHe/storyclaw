# 分场规划完整范例（Format Examples）

本文件是 `scene-optimizer` 技能的配套范例，给出**每个板块填满后的样子**与**完整输出模板**。SKILL.md 规定了结构，这里示范如何把一段故事梗概落成「逐场创作指导」。

> 核心红线：这是**写剧本之前**的一步。不要写对白、不要写动作行。只写描述性概述、分析戏剧功能、标注信息流。

---

## 输入梗概（示例）

> 面包店老板娘阿珍发现隔壁刚搬来的房客老周，正是十年前撞死她丈夫后逃逸的司机。阿珍隐忍接近，准备亲手报复；但相处中她发现老周一直在匿名资助一个孤儿——正是当年车祸中幸存的孩子。阿珍最终在能下手的那一刻收手，把真相告诉了警方。

下面把它拆成场次。范例展示其中 **3 场**（开场过渡场 + 两场重点场），并给出全局总览。真实任务应覆盖全部场次。

---

## 场次切分依据回顾
任一项改变即开新场：**地点 / 时间（含有明确戏剧断点的小跳跃）/ POV 人物或故事线 / 新戏剧单元（新目标、新阻碍、新结果）**。

---

## 场 1（过渡场示例 —— 仅简要说明）

> 过渡场不做五板块完整规划，一两句交代功能即可，避免规划文档臃肿。

**场 1 · 面包店清晨**｜日／面包店
搬家货车停在隔壁。阿珍隔着橱窗看见老周下车——一个侧脸的钩子。**功能**：把两人放进同一空间，埋下「似曾相识」的一句话坑（observation hole），为场 2 的认出蓄势。**信息**：观众与阿珍同步只得到「新邻居来了」，不知其身份。

---

## 场 2（重点场 —— 五板块完整规划）

### Scene Setting（场景设定）
- **场次与标题**：场 2 · 旧照片
- **时间·地点**：夜 / 阿珍的卧室
- **在场人物**：阿珍（POV）

### Content Summary（内容概述）
开场状态：阿珍因白天那张侧脸心神不宁。她翻出丈夫的遗物盒，找当年交警给的事故卷宗复印件。她比对卷宗里一张监控翻拍的模糊司机照片与白天的记忆。关键动作顺序：翻盒 → 找出照片 → 走到窗边看向隔壁亮着的灯 → 回头再看照片。转折点：她认出来了——耳后那颗痣。收尾状态：从「不安」变为「冰冷的确定」，她吹灭了一直为丈夫点着的那盏小灯。情绪弧：思念→怀疑→确认→杀意初生。承重视觉元素：那盏长明的小灯、耳后的痣。

### Function Analysis（功能分析，按重要性排序）
1. **【最重要】揭示** — 通过痣的比对，让阿珍（与观众）确认老周即逃逸司机，确立全片的故事情境。
2. **人物塑造** — 她吹灭长明灯这一动作，把「隐忍」与「决意」一次说清。
3. **情节铺垫** — 事故卷宗与照片为后续「她掌握证据、却选择私刑」埋线。
4. **情绪转折** — 完成从思念到杀意的转向，为后续接近行动提供动机引擎。

### Dramatic Points（戏剧点）
- 冲突位置：中后段（认出痣的瞬间）。
- 冲突类型：角色 vs 命运（仇人竟是邻居）+ 自我（隐忍 vs 复仇）。
- 此刻各方所求：阿珍想确认真相，又怕真是他。
- 张力峰值与释放：峰值在「比对成功」的定格；释放为吹灯——不是泄压，而是把压力转成冷决心（符合压力只增不减）。
- 赌注：若她认错人，将错杀无辜；若认对，她将走上不归路。

### Information Flow（信息流）
- **观众得知**：老周高度可能就是逃逸司机；阿珍已知情且起意。
- **观众被引导相信**：老周是纯粹的恶人（此处刻意让观众与阿珍同仇——为后半段「资助孤儿」的反转蓄误导；公平性：本场不给任何老周善意的线索）。
- **角色得知**：阿珍确认身份。
- **悬念／保留**：老周是否知道阿珍是谁？老周这十年在做什么？（全部保留）

## 场 3（重点场 —— 五板块完整规划）

### Scene Setting
- **场次与标题**：场 3 · 多出来的一份
- **时间·地点**：日 / 面包店后厨
- **在场人物**：阿珍（POV）、老周

### Content Summary
开场状态：阿珍已决意接近老周，借口收房租上门，邀他到店里帮忙搬面粉。关键动作：老周笨拙地搬货、付钱时多塞了一份；阿珍冷眼观察，寻找下手的时机与方式。关键交流：老周提到「每月要寄一笔钱，不能断」，含糊其辞；阿珍追问，他回避。转折点：老周走后，阿珍在他落下的外套里摸到一张汇款单存根，收款人是某儿童福利院。收尾状态：阿珍的「冷」第一次出现裂缝——她预设的「纯恶人」与眼前这个寄钱的人对不上。情绪弧：算计→不耐→困惑。承重视觉：汇款单存根、那多出来的一份钱。

### Function Analysis（按重要性排序）
1. **【最重要】误导观众的松动 / 揭示的第一道缝** — 汇款单第一次给出「老周或许另有故事」的反向信息，开始拆解前一场建立的「纯恶人」预设。
2. **关系转变** — 两人从「猎人与猎物」被迫进入日常相处，关系开始复杂化。
3. **人物塑造** — 老周的笨拙、多给的钱、回避的神态，立体化他的「不像逃犯」。
4. **节奏调节** — 在场 2 的高压后给一段「日常」呼吸，但暗流不断。

### Dramatic Points
- 冲突位置：贯穿全场（表面平静，内里阿珍在找时机、老周在藏秘密）。
- 冲突类型：信息冲突（阿珍知道他是谁、他不知道她是谁；他藏着汇款、她在挖）。
- 此刻各方所求：阿珍要接近并确认动手方式；老周要维持平静、守住他的秘密。
- 张力峰值与释放：峰值在「摸到汇款单」；不释放，转为新的悬念（他为谁寄钱）。
- 赌注：阿珍的复仇决心是否会被这条反向信息动摇。

### Information Flow
- **观众得知**：老周每月匿名给某儿童福利院汇款。
- **角色得知**：阿珍得知汇款一事（但尚不知收款孩子的身份）。
- **悬念／保留**：钱寄给谁？是否与车祸有关？老周是否在赎罪？——保留到后段揭示。

## 全局总览（Global Overview）

### Scene Function Chain（功能链条图）
```
场1 似曾相识(面包店) → 场2 揭示·确认仇人(卧室) → 场3 反向信息·第一道缝(后厨)
        埋观察坑              确立故事情境/起杀意         松动「纯恶人」预设
   ↓（后续）场4 接近升级 → 场5 资助真相揭晓 → 场6 能下手的一刻收手 → 场7 报警·终结
```

### Emotional Curve（情绪曲线）
低温起步（场1 不安）→ 陡升至冰点（场2 杀意，热）→ 维持高压但插入日常（场3 困惑，半热）→ 后续在「决意」与「动摇」间拉锯，至场6 触顶后骤冷收束。标注：场2 为第一个情绪高峰，场3 是第一处「呼吸但不松手」的半热场。

### Information Release Rhythm（信息释放节奏）
- 场1 末：观众知「来了新邻居」。
- 场2 末：观众知「邻居=逃逸司机，阿珍已起意」——**第一次大揭示**。
- 场3 末：观众知「老周在匿名资助福利院」——**第一道反向信息**，误导开始被回收。
- 误导设计：场2 刻意让观众与阿珍同仇、不给老周善意线索（公平埋伏）；场3 起逐场松动，确保最终反转时观众是「我早该看出」而非「凭空冒出」。

### Creative Notes（创作备注）
- 「长明灯」与「耳后痣」是跨场承重符号，建议全片复现（灯=思念/决意的开关，痣=身份铁证）。
- 故意留白：老周是否认得阿珍，全片保留到接近结尾才点破，维持双向悬念。
- 可选方向：场3 的汇款单也可改为阿珍「无意听到」而非「摸到」，若想弱化她的主动算计、强化偶然——取舍点在于要不要保住「阿珍始终是清醒的复仇者」这一人物基调（建议保留主动版）。
- 视觉/转场提示：场2→场3 用「吹灭的灯」叠化到「清晨面包店的烤箱火」，冷决心接日常烟火，形成反差。
