This skill produces a structured scene-by-scene planning document from a story synopsis — a blueprint that a dialogue-writing agent can later use to write the actual screenplay.

## Core Principle

This is a PRE-WRITING step. Do NOT write screenplay dialogue or action lines. Write descriptive summaries, analyze dramatic functions, and map out information flow. Output must answer, for every scene:
1. What happens? (descriptive summary)
2. Why is it here? (dramatic functions, ranked by importance)
3. Where is the heat? (key dramatic points, conflict, tension)

## Scene Division Rules

A new scene begins when ANY of these change:
- Location / setting
- Time (including short jumps with a clear dramatic break)
- Point-of-view character or storyline
- A new dramatic unit begins (new objective, obstacle, or outcome)

## Scene Planning Format

For each scene, produce these sections:

### Scene Setting
- Scene number and title (a punchy label)
- Time of day and location
- Characters present (list all, mark POV character if applicable)

### Content Summary (内容概述)
Write a descriptive summary of what happens. Use prose, not screenplay format. Cover:
- Opening state and closing state (what changed?)
- Key actions: who does what, in what order
- Key exchanges: who talks to whom, what information passes between them
- Emotional arc: whose feelings shift, at what moment, and why
- Load-bearing visual or atmospheric elements

Guidelines: Be concrete, not vague. Include the turning point. One to two paragraphs.

### Function Analysis (功能分析)
List ALL dramatic functions this scene serves. For each, write one sentence explaining how it's achieved. Then **rank them** and mark the single most important one.

Function types to consider:
- **情节铺垫** — Sets up a future event, plants a clue
- **信息传递** — Conveys key information to audience/characters
- **人物塑造** — Reveals character through action, choice, or reaction
- **情绪建立/转折** — Establishes or pivots emotional tone
- **冲突升级/引爆** — Escalates existing or triggers new conflict
- **揭示** — A character (or audience) learns something that changes understanding
- **误导观众** — Deliberately leads audience toward a false conclusion. Must be fair — when the truth is revealed, the audience should think "I should have seen that" not "that came from nowhere".
- **关系建立/转变** — Defines or redefines relationships
- **主题深化** — Reinforces or complicates the central theme
- **节奏调节** — Provides pacing relief or builds tension

### Dramatic Points (戏剧点)
- Where does the conflict live? (early / middle / late)
- Conflict type: 角色 vs 角色 / 自我 / 环境 / 命运 / 未知
- What each side wants in this moment
- Peak tension moment — where does it peak and how does it release?
- Stakes: what if the conflict is not resolved?

### Information Flow (信息流)
- **观众得知**: what new information the audience receives
- **观众被引导相信** (if this scene employs misdirection): what false conclusion is being steered toward, through what means
- **角色得知**: which characters learn what (be specific per character)
- **悬念/保留**: what remains hidden, and from whom

## Global Overview

After planning all scenes, produce:

### Scene Function Chain
Visual diagram showing how each scene's primary function connects to the next. Include location indicators (e.g., 凶宅/面包店).

### Emotional Curve
Describe the emotional rhythm across all scenes. Mark peaks, valleys, turning points. Note "hot" (high tension) vs "cool" (breathing room) scenes.

### Information Release Rhythm
Track how information is parceled out across scenes. What does the audience know at each scene's end that they didn't know before? Where are the major reveals? Track misdirection if used.

### Creative Notes
- Creative choices made during planning (and why)
- Gaps deliberately left unfilled
- Alternative directions worth considering
- Notes on visual design, scene transitions, etc.
