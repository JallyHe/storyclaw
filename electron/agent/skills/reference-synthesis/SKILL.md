# 资料整合

**定义**：资料整合把零散的原著、背景资料、采访素材提炼整合为结构化的创作输入（人物档案、时间线、设定要点、可用桥段）。

**核心约束**：
- 按"人物/事件/设定/可用桥段"分类归档，去重去噪。
- 保留出处，标记哪些可直接用、哪些需改编。
- 产出要能直接喂给后续创作专家。

**在创作中的应用**：资料研究专家用来产出结构化资料包。

**自检清单**：
- [ ] 分类归档
- [ ] 去重去噪
- [ ] 标可用/需改编
- [ ] 可直接喂给创作
