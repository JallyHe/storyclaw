# 先挖坑再填坑

**定义**：先挖坑再填坑：任何被解释、被回收的细节，都必须先在更早的位置出现（挖坑），再在后面解释（填坑）。适用于从全篇结构到单句台词的所有尺度。

**核心约束**：
- 禁止「无中生有」的解释——填坑前必须有对应的坑。
- 坑要埋得自然，不能让观众一眼看穿是伏笔。
- 重要的填坑前应有多次铺垫；越大的反转，坑挖得越早越多。

**在创作中的应用**：阶段2 规划伏笔、阶段3-5 落地、阶段6 检查回收闭环；维护一张「挖坑/填坑总表」。

**自检清单**：
- [ ] 每处解释都有更早的坑
- [ ] 坑的出现自然
- [ ] 重要反转有足够铺垫
- [ ] 挖坑/填坑成对闭环
